# Vaquero Connect

Vaquero Connect is a social platform designed to help UTRGV students create, discover, and participate in campus events and activities.

## Project Overview

Vaquero Connect is a student-oriented social application that connects individuals to campus activities that align to their interests, majors, and clubs. The platform enables users to browse upcoming events, manage their schedules, and engage with their community across various campus organizations.

## Core Features

### User Profiles

- Personalized user accounts with profile management

### Event Management

- **Create/Update/Delete Events:** Club officers and campus organizations can post/update/delete events
- **Browse Events:** Students can discover events filtered by categories
- **RSVP System:** Users can register for events with capacity limits

### Event Categories

- **Academic:** Study groups, seminars, guest lectures, tutoring
- **Sports:** Intramural games, club sports events, collegiate games
- **Social:** Club meetings, gatherings, networking events, job fairs
