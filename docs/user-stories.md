# User Stories

## User Management
1. As a student, I want to create an account, so that I can access Vaquero Connect and engage with the campus community.
2. As a registered user, I want to log in and out securely, so that my account remains private and protected.
3. As a registered user, I want to update my profile picture, username, and bio, so that other students can identify me on the platform.
4. As a registered user, I want to change my password, so that I can keep my account secure.

## Posts / Feed
5. As a logged-in user, I want to create posts on the homepage feed, so that I can share what's happening with the campus community.
6. As a logged-in user, I want to edit and delete my own posts, so that I can correct mistakes or remove outdated content.
7. As a visitor, I want to view posts on the homepage, so that I can see what other students are talking about on campus.

## Events
8. As a club officer, I want to create, edit, and delete campus events, so that students are informed about upcoming activities.
9. As a student, I want to browse events filtered by category (Academic, Sports, Social, Emergency), so that I can find events relevant to my interests.
10. As a student, I want to RSVP to events and cancel my RSVP, so that organizers know how many people are attending and I can manage my schedule.
